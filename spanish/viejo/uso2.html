Tema 4: conoer, saber, poder

5 oraciones para escribir: [prueba]{ - ¿Conoces al marido de Carmen? - Vamos, conozco un sitio donde ponen unas tapas buenisimas. - Hoy no puedo corer mucho, me he hecho daño en un pie y me duele un poco. - Gracias a Internet, hoy las noticias {se conocen|se saben} inmediatamente en todo el mundo. - ¿Sabes hablar árabe? } ### Tema 5: estuve/estaba/he estado + gerundio ### Tema 7: Las preposiciones a, con, de, en, hacia, hasta, para, por 1. [ A ] se usa para explicar: - destino: a Madrid - hora: a las una - objetivo / finalidad / complemento indirecto: voy a ver / a usted - complemento diercto para persona: a los invitados - precio / fecha / distancia / temperatura, con el verbo «estar»

  1. [ con ] se usa para explicar:
    • compañía: Me quedé en casa con mi familia.
    • instrumento / modo:
      • Mi hijo solo hace fotos con el móvil.
      • Hizo los ejercicios con rapidez.
  2. [ de ] se usa para explicar:
    • posesión: ¿De quién es este libro? ¿Es de Beatriz?
    • material: de lana
    • origen o procedencia: de Taichung
    • tiempo / momento del día: de lunes a viernes, a las 6 de la tarde
    • modo: Siempre duerme de lado.
    • descripción / profesión, con el verbo «estar»
  3. [ desde ] se usa para explicar:
    • Origen en el tiempo y en el espacio: desde Taipei / desde 1980
  4. [ en ] se usa para explicar:
    • lugar (ubicación): Está en el dormitorio, en la cama.
    • tiempo: Nació en 1980.
    • medio de transporte: Siempre vengo en bicicleta, es más ecológico.
  5. [ hacia ] se usa para explicar:
    • dirección / lugar: Los bomberos ya van hacia allí.
    • tiempo (no muy preciso): No sé, pero creo que vi mi primera exposición de Picasso hacia 1956.
  6. [ hasta ] se usa para explicar:
    • límite en el tiempo o el espacio: Te esperaré hasta las 5. Luego me iré.
  7. [ para ] se usa para explicar:
    • objetivo / finalidad: no sirve para nada / para ser ingeniero
    • dirección / lugar: Ya que vas para la cocina, llévate esto, por favor.
    • tiempo: Limpiáme este traje para mañana / para el mes que viene tengo tres conciertos
  8. [ por ] se usa para explicar:
    • causa: Vamos a brindar por el éxito del disco. / Lo han despedido clases por vago.
    • lugar: El gato debió de entrar por la ventana. / ¿Te gusta pasear por el campo?
    • complemento agente en la voz pasiva: El concierto ha sido dirigido por Luis Cobos.
    • precio: ¡Qué barato! Me he comprado una camisa por 6 euros.
