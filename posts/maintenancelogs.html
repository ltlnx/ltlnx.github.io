網站維護流水帳

一月三十一日

　　因為這是第一天打流水帳，所以打得詳細一點。

　　之前這個網站用所謂的「靜態網頁產生器」產生部落格，後來打掉重練，逐字寫（從網路各處複製）出現在這個樣子。建置細節如下。

一、網頁本體（ＨＴＭＬ）

　　大多是基於小學電腦課的回憶跟Google大神。這部分沒什麼特別。

二、點綴（ＣＳＳ）

  • 字體：因為想有直書印刷的感覺，標題選用黑體，內文選用宋體；但網頁前端工程師常遇到的問題，就是沒字體可用，本人選擇放棄掙扎。
  • 主題：之後應該會再改。
  • 序號排列：這個比較好玩，值得特別標出來寫。

    　　這個部落格第一個網頁就是「研究生評論」。在稍微修改完格式之後，我將網頁連結貼到ＩＧ上。朋友看了反映，1. 2.等編號跟英文一起被轉了九十度，如果能轉回來變成橫的，會更好看。於是我找到時間，著手進行改造。

    　　首先參考了Josh Comeau的網站，裡頭寫道：

    1. ＣＳＳ無法控制編號的樣式，頂多規定要從哪一號開始、要順著數還是倒著數，還有padding、margin等參數。
    2. 但有個東西叫「Counter」，它是個能記錄號碼的變數，我們只要呼叫它就可以顯示目前的數值。
    3. 接下來只要在ＣＳＳ的「ol li:before」中呼叫Counter，就可以套格式了！

    　　在複製那段ＣＳＳ過來後，一開始想說直接套上「writing-mode: horizontal-tb」（書寫方向：橫向）就好了，畢竟這個參數就是讓整個網站成為直書的秘辛。這個方法在火狐中奏效，在Chrome與Safari中就沒那麼幸運了。因此必須找比較通用的辦法。

    　　接下來想到了要旋轉文字，參考css-tricks的介紹，終於成功了。以下是最後試成功的ＣＳＳ：

    ol {
    list-style: none;
    counter-reset: cupcake;⬅ 重設Cupcake計數器
    }
    ol li {
    counter-increment: cupcake;⬅ 設定一個計數器，命名為Cupcake
    }
    ol li:before {⬅ 神奇的before參數（請見參考網站）
    content: counters(cupcake, '.') '.';（詳見補充）
    position: absolute;⬅ 設定為絕對位置，很重要
    margin-top: -1.2em;⬅ 為了讓排版不跑掉
    transform: rotate(270deg);⬅ 轉回橫的
    }
    /* 如果要拿去用記得把註解刪掉！ */

    補充

    • 第一個「'.'」是數字跟數字之間要填充的東西，
      以「二之一之二」為例：
      • 如果是句點就會顯示「2.1.2」；
      • 如果是連字號就會顯示「2-1-2」
    • 第二個「'.'」是編號之後要顯示的東西，
      一樣以「二之一之二」為例：
      • 如果是句點就會顯示「2.1.2.」；
      • 如果是冒號就會顯示「2.1.2:」
