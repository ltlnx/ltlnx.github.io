如何編譯 Libchewing
與 Fcitx-chewing

從Fcitx的Google Groups，「彭聖嚴」的留言複製過來的，以便往後查閱。

---以下為原留言

系統: 全新灌在 VMware Workstation 12 player 的 lubuntu-14.04.4-desktop-amd64


  • 安裝完＆軟體自動更新
    sudo apt-get install fcitx fcitx-libs-dev
  • 切換鍵盤輸入法系統 從 ibus => fcitx ( 右上角齒輪 > 系統設定值 > 語言支援 ）
  • 登入再登出
  • sudo apt-get purge ibus-chewing
  • sudo apt-get autoremove // 移除 libchewing3
  • sudo apt-get install autoconf automake libtool cmake libclang-dev
  • sudo apt-get install build-essential libsqlite3-dev cmake git pkg-config
  • sudo apt-get install texinfo

libchewing installation （不要用cmake! )

  • git clone https://github.com/chewing/libchewing.git
  • cd libchewing
  • ./autogen.sh
  • ./configure --prefix=/usr
  • make
  • sudo make install

fcitx-chewing installation

  • git clone https://github.com/fcitx/fcitx-chewing.git
  • cd fcitx-chewing
  • mkdir build; cd build
  • cmake ..
  • make
  • sudo make install

點擊右上角小鍵盤 > 重新啟動 應該就可以在右上角小鍵盤 > 輸入法 看到新酷音了（灑花）
